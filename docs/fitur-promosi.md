# Tabel List Fitur AtigaCBT (Media Promosi)

| Kategori | Fitur Utama | Nilai Promosi / Manfaat |
|---|---|---|
| Manajemen Pengguna | Login JWT + RBAC (Admin, Guru, Siswa) | Akses aman dan terkontrol sesuai peran pengguna |
| Dashboard | Dashboard Admin, Guru, Siswa dengan statistik real-time | Monitoring kondisi pembelajaran dan ujian lebih cepat |
| Bank Soal | CRUD bank soal + Quick Add batch | Pembuatan soal lebih efisien dan hemat waktu |
| Import Soal | Import soal dari DOCX + preview parsing sebelum simpan | Migrasi soal lama ke sistem digital jadi praktis |
| Manajemen Ujian | Buat jadwal ujian, target peserta, sesi, dan token | Pelaksanaan ujian terstruktur dan mudah dikontrol |
| Pelaksanaan Ujian Siswa | Join ujian, kerjakan ujian dengan autosave | Mengurangi risiko jawaban hilang saat koneksi bermasalah |
| Monitoring Ujian | Monitor sesi & peserta (progress/join/online) + SSE realtime | Proktor dapat memantau ujian langsung tanpa refresh |
| Penilaian & Analitik | Auto-score, distribusi nilai, analisis butir | Evaluasi kualitas ujian lebih objektif dan berbasis data |
| Koreksi Essay | Fitur koreksi essay + feedback | Mendukung asesmen komprehensif, termasuk soal uraian |
| Laporan | Export hasil ujian & analitik ke Excel multi-sheet | Pelaporan sekolah lebih rapi dan siap presentasi |
| Pengumuman | Pengumuman tersegmentasi (level/group/siswa) | Komunikasi akademik lebih tepat sasaran |
| Verifikasi Pendaftaran | Verifikasi pendaftaran + bulk approve + set pending | Proses administrasi penerimaan siswa/guru lebih cepat |
| Master Data | CRUD program, level, kelas, mapel, guru, siswa + import Excel | Operasional akademik terpusat dalam satu aplikasi |
| PWA & UX | Bisa diinstal sebagai PWA + Fullscreen mode ujian | Pengalaman ujian modern di desktop maupun mobile |
| Stabilitas Sistem | Global error handling, toast notifikasi, caching TanStack Query | Aplikasi terasa lebih cepat, stabil, dan nyaman dipakai |
| Keamanan & Audit | Request ID, structured logging, audit log aksi mutasi | Jejak aktivitas terdokumentasi untuk akuntabilitas |
| Maintenance | Backup & restore database | Keamanan data lebih terjaga untuk operasional jangka panjang |
